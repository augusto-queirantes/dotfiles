# Dotfiles

That's my dotfiles

## Installation

1. Execute the installation script

`./install.sh`

2. Reboot your computer

3. Select `FiraMono Nerd Font` as your terminal font (remember to set your preferred font size, mine is 16)

4. Install the node version you want using `nvm install <version>`. Remember to set node version on path.

5. Install the ruby version you want using `rvm install <version>` and `rvm use <version>`. Remember to set ruby version on path.
