# Neovim

## Post configuration

1. Verify that node, ruby and gems are set in the bash path
2. Check that the LSP servers you want are working. To do this run `:LspInfo` on vim while the desired lang file are opened
3. Execute `:checkhealth` to verify possible erros during vim installation

### Ruby related

1. `gem install neovim`
2. `gem install --user-install solargraph`

### Node related

1. `npm install -g neovim`
